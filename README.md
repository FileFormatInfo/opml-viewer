# Sitemap Viewer [<img alt="Sitemap Viewer logo" src="public/favicon.svg" height="90" align="right" />](https://view.sitemap.style/)

This is a graphical sitemap viewer for [Sitemap.Style](https://www.sitemap.style/).

## Running locally

```
./run.sh
```
